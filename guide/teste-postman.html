# Executar a API no Postman

# O que é o Postman?

O Postman (opens new window) é um API Client que facilita aos desenvolvedores criar, compartilhar, testar e documentar APIs. Isso é feito, permitindo aos usuários criar e salvar solicitações HTTP e HTTPs simples e complexas, bem como ler suas respostas.

# Como executar a API no Postman

O primeiro passo para executar o projeto é instalando o node_modules na pasta do projeto e usando o comando npm install no terminal do projeto. Feito isso, basta executar o comando node index.js que é responsável por rodar o projeto no node.

O próximo passo é importar o a coleção do link: https://www.getpostman.com/collections/7a32513c472e5f36a633 (opens new window)

Ao concluir os passos anteriores, é esperado que no Postman exiba a seguinte coleção:

  • GET Listar animais
  • GET Listar animais no local
  • GET Listar locais
  • POST Cadastrar animal
  • POST Cadastrar local
  • PUT Alterar animal
  • PUT Altera local
  • REMOVE Remover animal
  • REMOVE Remover local

Após importado, basta testar a API.
