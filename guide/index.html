# Introdução

Esta documentação tem como finalidade auxiliar o uso da API da aluna Francisca Natália Simões de Araújo, na matéria de Programação Web.

A API foi baseada no jogo de videogame Stardew Valley (opens new window), que pertence ao gênero RPG e simulação e foi desenvolvido por Eric Barone. O jogo simula a vida no campo, onde o jogador herda uma fazenda e pode fazer inúmeras atividades para obter dinheiro e conseguir expandir/construir mais na sua fazenda.

Uma das atividades para conseguir dinheiro é a venda de animais e/ou venda de recursos derivados de animais, a API foi baseada neste ponto, onde ao comprar os animais, o jogador deve escolher o local onde este animal será colocado (de acordo com seu tipo, ex: Vaca fica no celeiro e galinha no galinheiro). Foram criadas as tabelas “animal” e “local”, onde cada animal pertence a um local e cada local pode ter vários animais, fazendo a relação 1:N.
